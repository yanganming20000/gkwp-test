# GKWP 幸福测评系统 - 部署说明

## 第一步：注册 MongoDB Atlas（免费数据库）

1. 访问 https://www.mongodb.com/cloud/atlas
2. 点击「Try Free」注册账号（用邮箱注册即可）
3. 创建免费集群（选 M0 Free Tier，选新加坡节点）
4. 创建数据库用户，记住用户名和密码
5. 在 Network Access 里点「Add IP Address」→「Allow Access from Anywhere」
6. 点「Connect」→「Connect your application」，复制连接字符串
   格式：mongodb+srv://用户名:密码@cluster0.xxxxx.mongodb.net/gkwp

## 第二步：注册 GitHub（存放代码）

1. 访问 https://github.com 注册账号
2. 点「New repository」，名字填「gkwp-server」，选 Private（私有）
3. 把本项目所有文件上传到仓库

## 第三步：部署到 Render（免费服务器）

1. 访问 https://render.com 用 GitHub 账号登录
2. 点「New +」→「Web Service」
3. 选择你的 gkwp-server 仓库
4. 配置如下：
   - Name: gkwp-server
   - Environment: Node
   - Build Command: npm install
   - Start Command: node server.js
5. 在「Environment Variables」添加：
   - MONGODB_URI = 你的MongoDB连接字符串
   - ADMIN_PASSWORD = 你设置的管理员密码（建议用强密码）
6. 点「Create Web Service」，等待部署完成
7. 部署完成后会得到域名，如：https://gkwp-server.onrender.com

## 第四步：日常使用

### 管理后台
访问：https://你的域名.onrender.com/admin.html
输入管理员密码登录

### 生成验证码
1. 登录管理后台
2. 填写生成数量（如：1）
3. 填写备注（如：付款人微信名）
4. 点「生成验证码」
5. 把生成的验证码（如：ABC12345）发给付款用户

### 用户使用流程
1. 用户付款19.9元
2. 截图发给你
3. 你在管理后台生成一个验证码发给用户
4. 用户打开网站输入验证码进入测评
5. 每个验证码只能用一次，用完自动失效

## 注意事项

- Render 免费版服务器15分钟无访问会休眠，第一次访问会慢10-30秒（正常现象）
- 如需避免休眠，后期可升级到 Render 付费版（约$7/月）
- MongoDB Atlas 免费版有512MB存储，够存几万条记录
- 定期在管理后台查看使用记录，确认没有异常
